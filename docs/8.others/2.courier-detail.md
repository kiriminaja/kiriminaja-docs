# Courier Detail

The list of couriers available in the KiriminAja API, staging and production may be the difference

```shell
[POST] /api/mitra/courier_services
```

### Request

| Field          | DataType | Nullable | Desc    |
| -------------- | -------- | -------- | ------- |
| `courier_code` | string   | `false`  | Courier |

```json
{
  "courier_code": "jne"
}
```
