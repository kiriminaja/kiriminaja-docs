# Cancel Order Express

Cancel your instant delivery shipment requests

```shell
[DELETE] /api/mitra/v4/instant/pickup/void/{order_id}
```

## Request Example

```bash
curl --location 'https://tdev.kiriminaja.com/api/mitra/v4/instant/tracking/NBL2-1705636897084' \
--header 'Accept: application/json' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {token}'
```
