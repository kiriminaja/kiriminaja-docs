# Province

The list of provinces that we support cannot be compared to expeditions. You don't need to do manual mapping to the expedition

```shell
[POST] /api/mitra/province
```

## Response Sample

```json
{
  "status": true,
  "method": "provinsis",
  "text": "Berhasil",
  "datas": [
    {
      "id": 1,
      "provinsi_name": "Bali"
    },
    {
      "id": 2,
      "provinsi_name": "Bangka Belitung"
    }
  ]
}
```
